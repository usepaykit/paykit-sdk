PayKit provides a unified API that works across all payment providers. This reference covers the core resource types.

## Types

<br />

```typescript
type PaykitMetadata = {
  [key: string]: string;
};
```

## Customer

<br />

```typescript
interface Customer {
  id: string;
  email?: string;
  name?: string;
  metadata?: PaykitMetadata;
}

interface CreateCustomerParams {
  email: string;
  name?: string;
  metadata?: PaykitMetadata;
}

interface UpdateCustomerParams {
  email?: string;
  name?: string;
  metadata?: PaykitMetadata;
}
```

## Checkout

<br />

```typescript
type BillingMode = 'one_time' | 'recurring';

interface CreateCheckoutParams {
  customer_id: string;
  item_id: string;
  session_type: BillingMode;
  metadata: PaykitMetadata;
  provider_metadata?: Record<string, unknown>;
}

interface RetrieveCheckoutParams {
  id: string;
}

interface Checkout {
  id: string;
  customer_id: string;
  payment_url: string;
  metadata: PaykitMetadata | null;
  session_type: BillingMode;
  products: Array<{ id: string; quantity: number }>;
  currency: string;
  amount: number;
}
```

## Subscription

<br />

```typescript
type SubscriptionStatus = 'active' | 'past_due' | 'canceled' | 'expired';

interface Subscription {
  id: string;
  customer_id: string;
  status: SubscriptionStatus;
  current_period_start: Date;
  current_period_end: Date;
  metadata: PaykitMetadata;
}

interface UpdateSubscriptionSchema {
  metadata: PaykitMetadata;
}
```

## Invoice

<br />

```typescript
interface Invoice {
  id: string;
  amount: number;
  currency: string;
  metadata: PaykitMetadata;
  customer_id: string;
  billing_mode: BillingMode;
}
```

## Webhook Events

PayKit supports the following webhook events across all providers:

- `$customerCreated` - Triggered when a customer is created
- `$customerUpdated` - Triggered when a customer is updated
- `$customerDeleted` - Triggered when a customer is deleted
- `$subscriptionCreated` - Triggered when a subscription is created
- `$subscriptionUpdated` - Triggered when a subscription is updated
- `$subscriptionCancelled` - Triggered when a subscription is cancelled
- `$checkoutCreated` - Triggered when a checkout is created
- `$invoicePaid` - Triggered when an invoice is paid
