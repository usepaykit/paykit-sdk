## Overview

Add a PayKit block with shadcn, then follow your backend guide.

## Install via Registry

<br />

```bash
npx shadcn@latest add https://usepaykit.dev/r/stripe-nextjs
```

## Pick your backend

- [Next.js](/docs/nextjs-installation)
- [Hono](/docs/hono-installation)
- [Express](/docs/express-installation)

## Client (optional)

If you’re building a Next.js UI, see [client side usage](/docs/concepts/client-side-usage/) for hooks and provider setup.
