We shipped the initial public preview:

- Core SDK: unified API for checkouts, customers, subscriptions, payments, refunds, webhooks.
- Providers: Stripe and PayPal, with Polar and GoPay in progress.
- React package: context and hooks for common flows.
- UI kit: Tailwind-based primitives for consistent UX.
- Registry: shadcn-compatible blocks for Next.js/Hono, with ORM variants.
